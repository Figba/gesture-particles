# Figma 设计还原项目

这是一个根据提供的 Figma 链接还原的网页设计项目。

## 包含的文件
- `index.html`: 主网页文件，包含了所有的结构和样式。

## 如何查看
1. 直接双击打开 `index.html` 文件，或者将其拖拽到浏览器中即可查看效果。
2. 项目使用了 Tailwind CSS (通过 CDN 引入)，所以需要保持网络连接以加载样式库。
3. 图片资源使用了占位图 (Placeholders)，因为无法直接从设计稿下载原图。

## 代码说明
- **结构**: 使用了 HTML5 语义化标签。
- **样式**: 使用了 Tailwind CSS 框架，这是一种流行的实用优先 CSS 框架，允许在 HTML 类名中直接写样式。
- **注释**: 代码中包含中文注释，解释了各个部分的功能（如“工具栏”、“左侧主内容区域”、“评论区”等）。

## 注意事项
- 设计稿中的图片和图标已用占位符或相似的 SVG 图标替代。
- 界面上的红色圆点（悬浮标记）使用了绝对定位，可能在不同屏幕尺寸下位置会有所偏移，这是为了还原设计稿中的固定坐标。
